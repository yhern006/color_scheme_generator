# Color Scheme Generator

A color scheme generator where the user picks a color and a scheme type (monochrome, analogic, complement, etc.) and the generator will give you a color scheme from the user input.

## About
Solo Project from Scrimba's APIs and Async JS course in the Full Stack Developer Path.

This project is meant to get practice with APIs.

## Project Requirements
- [Completed] Follow Figma Layout
- [Completed] Make request to the Color API to get a color scheme
- [Completed] Display the scheme colors & hex values
- [Completed] Stretch goal: click hex values to copy to clipboard
- [Completed] Personal goal: make it responsive :)
